# Full-Stack Login Simulation

This is a full-stack web application that simulates a user authentication system. It features a React-based frontend with a login form and a success page, connected to an Express.js backend that handles login requests with hardcoded credentials for demonstration purposes.

## Tech Stack
- **Frontend:** React, Bootstrap, Axios
- **Backend:** Node.js, Express.js, CORS

## Features
- User login form with username and password fields
- Authentication simulation via API call to backend
- Responsive design using Bootstrap
- Success page displayed upon successful login
- Error handling for invalid credentials and connection issues

## Getting Started
1. Clone the repository
2. Install backend dependencies: `cd backend && npm install`
3. Install frontend dependencies: `cd frontend && npm install`
4. Start the backend server: `cd backend && npm start`
5. Start the frontend app: `cd frontend && npm start`
6. Open http://localhost:3000 in your browser
7. Login with username: `admin`, password: `1234`
